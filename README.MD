- ¿Para qué sirve el archivo package.json?

Es un archivo para proyectos de NodeJS o de js donde se declaran metadatos, las dependencias, scripts y configuracion de herra,oemtas

- ¿Qué diferencia notaron entre ejecutar un archivo directamente con Node.js y levantar el servidor de React?

La principal diferencia es que desde react podes realizar cambios, guardar el archivo y los cambios se ven reflejados en el servidor de forma instantanea, a diferencia de ejecutar un archivo con node donde una vez realizado los cambios, tenes que ejecutar manualmente el archivo
